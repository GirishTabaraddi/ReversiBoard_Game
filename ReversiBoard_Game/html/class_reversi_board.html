Detailed Description

A ReversiBoard Class.

ReversiBoard.h

Created on: 14-Nov-2023 Author: Girish

A class which representing a board with it's field being filled either by player 1 or player 2 color. Here 2 colors will be taken as Black, White, noPawn and option for the empty field. These states are represented in the form of enumeration. Also, the class has 2 methods which implement the logic of the game.
